# 🚀 Team Task Manager API

A full-stack backend application to manage projects and tasks with role-based access.

## 🔥 Features

* User Signup & Login (JWT Authentication)
* Project Management
* Task Assignment & Status Tracking
* Dashboard Analytics

## 🛠 Tech Stack

* Java Spring Boot
* MySQL
* JWT Authentication

## 🔗 APIs

### Auth

* POST /api/auth/signup
* POST /api/auth/login

### Projects

* POST /api/projects
* GET /api/projects

### Tasks

* POST /api/tasks
* GET /api/tasks

### Dashboard

* GET /api/dashboard/{email}

## ⚙️ Run Locally

1. Clone repo
2. Setup MySQL
3. Run Spring Boot app
